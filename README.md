# Cinch

A manager of large files.

From WordNet 3.0 (2006)

> **cinch**
>
> - **noun** any undertaking that is easy to do

## Install

Compile from source:

```sh
go get -d ./...
go build
```

Then move `cinch` to a directory in `$PATH`.

## Usage

```sh
cinch
```

Refer to DESIGN.md and source code (`cinch.go`) for more information.
